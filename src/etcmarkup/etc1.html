2.6.1 frame

frame은 하나의 화면에 여러 개의 html 페이지를 보여줄 수 있습니다. 물론 달일 페이지 구성도 가능합니다. frame은 사용자 임의로 크기 조절이 가능합니다.

    frameset cols="25%,27%"
    frame src="a.html"
    frame src="b.html"
    /frameset

frame은 닫기 태그가 없습니다. 여러분이 사용하시는 크롬의 개발자 도구도 위와 같이 frameset을 사용합니다.

frameset

frameset은 하나 이상의 frame요소를 포함하는 태그입니다. 속성으로는 cols와 rows가 있습니다.
cols: 화면을 가로로 분할 할 때 프레임 수와 크기를 지정합니다.
rows: 화면을 세로로 분할 할 때 프레임 수와 크기를 지정합니다.

noframes

noframes는 frame을 지원하지 않는 브라우저를 위해 존재합니다. frame을 지원하지 않는 브라우저일 경우 noframes안에 body태그의 내용을 출력합니다.

iframe

iframe은 페이지의 어느 곳에든 구성 가능합니다. 한 페이지에서 또 다른 페이지를 보여주고 싶을 때 사용합니다. iframe은 사용자 임의로 크기 조절이 불가능합니다.

2.6.3 주석

주석은 소스코드에 대한 설명을 명시해 놓는 것입니다. 소스코드에 주석처리를 하면 웹 브라우저가 인식하지 않으므로 소스코드 자체를 보류하는 용도로도 사용하기도 합니다. 주석처리를 하지 않는다면 유지보수가 어렵고 소스코드를 이해하기 어렵습니다.

2.6.4 meta

메타(meta)요소는 head요소 안에 존재하여 해당 페이지에 대한 정보를 나타냅니다. 이 정보에는 페이지가 어떤 내용을 담고 있고, 문서의 핵심이 무엇인지 나타냅니다. 이러한 정보는 검색포털에 검색엔진이 검색을 할 때에도 중요한 키워드가 되니 주의해서 작성해야 합니다. meta태그의 속성으로는 name과 content, http-equiv와 content가 일반적으로 같이 쓰입니다.

      meta name="description" content="홈페이지 설명"/>

name 속성 값으로는 주로 description, keywords, author가 사용됩니다.

name 속성 값
description 문서에 대한 설명
keywords 문서의 키워드
author 저자 이름

2.6.5 span

span은 대표적인 inline요소로 내용 중 일부분에 속성을 부여하고 싶을 때 주로 사용됩니다.


레이아웃을 나눌 때 설계가 가장 중요합니다.

2.7.1 블록 요소

블록요소는 한 개의 독립된 덩어리로 화면의 가로 폭 전체를 차지하는 요소들을 말합니다. 일반적으로 블록요소를 사용하면 라인이 바뀝니다. 블록요소는 블록요소와 인라인요소를 모두 포함할 수 있습니다. 대표적인 블록 요소는 article, header, nav, section, div가 있습니다.

2.7.2 인라인 요소

인라인 요소는 컨텐츠 자신의 크기 만큼 영역을 가지는 요소를 말합니다. 일반적으로 인라인 요소들은 인라인요소만 중첩이 가능합니다. 인라인 요소의 정렬은 텍스트 정렬의 형향을 받습니다. 대표적인 인라인 요소로는 span, strong등이 존재합니다

inline요소: 자신의 필요한 부분만 차지

block요소: 인라인 요소를 포함할 수 있으며 일반적으로 한 row를 차지합니다.
