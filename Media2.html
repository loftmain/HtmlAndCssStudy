2.3.1 img 삽입

img태그
img태그는 html 페이지에 이미지를 삽입할 때 사용하는 태그입니다. img는 'src'라는 필수 속성이 있어야 합니다.

src 속성은 브라우저에게 이미지 파일의 위치 및 파일명을 알려줍니다. 큰따옴표 안에 들어가는 경로는 절대경로이거나 상대경로이어야 합니다. alt 속성은 이미지가 보이지 않을 때 alt 속성에 적힌 텍스트를 이미지 대신 보여줍니다. title 속성은 사용자가 이미지 위에 마우스를 가져가면 title 속성에 적힌 텍스트를 툴팁으로 보여줍니다. 이미지의 크기를 지정할 때는 가로(width), 세로(height)에 원하는 폭을 픽셀 단위로 지정합니다.

2.3.2 img 배치

html 코드 내에 img 태그 위치는 이미지의 위치가 됩니다.

  1. 단락 위에 배치 : 이미지 아래에 단락이 시작합니다.
  2. 단락의 시작에 배치 : 이미지 하단에서 단락이 시작하게 됩니다.
  3. 단락 중간에 배치 : 단락 사이에 이미지가 배치됩니다.

서울의 시장의 모습을 담았다. 시간이 늦지 않았지만 주말이라 굳게 닫힌 상점의 문을 볼 수 있다.

2.3.3 audio

audio
audio는 음악 파일을 재생하기 위한 태그입니다.

src 속성은 브라우저에게 오디오 파일의 위치 및 파일명을 알려줍니다. src="폴더/파일명" controls 속성은 음악 컨트롤러를 불러옵니다. autoplay 속성은 자동으로 음악을 재생시킵니다. loop속성은 음악을 반복합니다.
